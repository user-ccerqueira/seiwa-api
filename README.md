# SEIWA API

API Procedimentos Médicos

## Inicialização

### Clonar projeto

```sh
git clone https://gitlab.com/digital-republic/gfx-api.git
```

### Create DB

```sh
yarn db:migrate
yarn db:seed
```

### Credenciais

```sh
Username: user-seiwa
Password: Seiwa@123
```

### Instalar dependências

```sh
yarn
```

## Utilização

```sh
yarn dev
```

## Swagger

```sh
url: http://localhost:1001/docs/
```
